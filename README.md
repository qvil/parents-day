# Parents Day

Page that share carnation to our thankful parents. 🙏

## Demo

![demo.gif](demo.gif)

## Prerequisites

- Node.js >= 10.x (Recommended the latest version)
- .env (Refer to [.env.example](.env.example))
- (Optional)Kakao developers app key refer to https://developers.kakao.com/

## Install

```
npm install
```

## Development

```
npm start
```

## Build & Deploy

```
npm run build
```

```
npm run prod
```
